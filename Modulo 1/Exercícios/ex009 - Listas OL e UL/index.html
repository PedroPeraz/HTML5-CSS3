Trabalhando com Listas

Listas Ordenadas

Com a tag "ol" é possível criar uma lista ordenada! E adicionar itens a partir da tag "li"

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Além disso podemos mudar o marcador a partir do parâmetro "type"!. E possuimos cinco tipos diferentes de marcadores

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Listas Não Ordenadas

Com a tag "ul" podemos criar listas não ordenadas. E assim como nas ordenadas podemos mudar os marcadores, com quatro opções diferentes.
