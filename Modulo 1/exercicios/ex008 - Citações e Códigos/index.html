Código-Fonte e Pré-Formatado

O comando document.getElementbyId('teste') é escrito em linguagem JavaScript.

É possível usar a tag "code" que deixa códigos em uma fonte monoespaçada, facilitando a leitura.

Caso queira mostrar linhas de código, pode ser colocado a tag "pre", que faz com que o código deixe de ficar assim:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR'') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Para ficar assim:

        
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR'')
else:
    print(f'O número {num} é ÍMPAR')
print('Fim do programa')

Citações

Para fazer uma citação, é possível usar a tag "q" que significa quote (citação em inglês).

Como diria o pai de um amigo: o computador é um burro muito rápido

Dessa forma, damos um sentido semântico a colocação das aspas.

Abreviações

Com a tag "abbr" é possível colocar o significado das abreviações inseridas no seu texto. Como por exemplo:

Estou estudando HTML e CSS, e estou adorando!

Texto Invertido
